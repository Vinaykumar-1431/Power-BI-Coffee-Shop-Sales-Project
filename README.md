# ☕ Coffee Shop Sales Dashboard | Power BI Project

## 📌 Project Overview
This project focuses on analyzing coffee shop sales data using Power BI to uncover meaningful business insights and support data-driven decision-making.

The dashboard provides a complete overview of sales performance, customer purchasing behavior, product trends, and peak business hours through interactive visualizations and KPI tracking.

---

## 🎯 Business Problem
Coffee shops generate large volumes of transactional data daily.  
The challenge is identifying:
- Peak sales hours
- Best-selling products
- Revenue trends
- Customer purchasing patterns
- Daily and monthly sales performance

This dashboard helps stakeholders monitor performance efficiently and improve operational decisions.

---

## 🛠️ Tools & Technologies Used
- Power BI
- DAX
- Power Query
- Excel/CSV Dataset

---

## 📊 Dashboard Features

### ✅ KPI Metrics
- Total Sales
- Total Orders
- Average Order Value
- Quantity Sold

### ✅ Sales Analysis
- Hourly Sales Trend
- Daily Sales Performance
- Monthly Revenue Analysis
- Peak Sales Hour Identification

### ✅ Product Insights
- Top Selling Products
- Category-wise Sales Analysis
- Revenue Contribution by Product

### ✅ Interactive Features
- Dynamic Filters & Slicers
- Drill-through Analysis
- Interactive Visualizations

---

## 📈 Key Insights
- Morning hours generated the highest sales revenue.
- Coffee beverages contributed the majority of total sales.
- Weekend sales were significantly higher compared to weekdays.
- Certain product categories consistently outperformed others in revenue generation.
